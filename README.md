## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation & Setup

1. **Backend Setup**

- cd superhero-api
- npm install
- npm run start:dev

2. **Frontend Setup**

- cd superhero-ui
- npm install
- npm run dev
